# 🚀 AutoGen Agents Course

This repository is a **complete hands-on learning journey** to master AI agents using Microsoft AutoGen. It covers everything from fundamentals to advanced multi-agent systems and a real-world capstone project.

---

## 🎯 What You'll Learn

* Fundamentals of AutoGen framework
* Working with different LLM providers (OpenAI, Groq, etc.)
* Agent communication patterns
* Tool-using AI agents
* Code execution with agents
* Multi-agent collaboration systems
* Memory and RAG-based agents
* Production-ready agent architectures
* Real-world AI automation projects

---

## 📂 Course Structure

```
module_01_foundations/          → Basics of AutoGen
module_02_llm_providers/        → LLM integrations
module_03_conversation_patterns/→ Agent communication design
module_04_tool_use/             → Tools & function calling
module_05_code_execution/       → Code-executing agents
module_06_group_chat/           → Multi-agent systems
module_07_memory_rag/           → Memory + RAG pipelines
module_08_projects/             → Hands-on projects
module_09_advanced/             → Advanced agent patterns
module_10_capstone/             → Final end-to-end system
```

## 🛠️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/dearnidhi/Autogen-Agents-Course.git
cd Autogen-Agents-Course
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Add API keys in `.env`:

```env
OPENAI_API_KEY=your_key_here
GROQ_API_KEY=your_key_here
```

4. Run modules using Jupyter Notebook or Python scripts.

---

## ⚠️ Important Notes

* Never commit `.env` file
* `venv/` should not be pushed
* This is a learning-first repository
* Each module builds on the previous one
